# Portfolio Hero 3D Cube Navigation

## コンセプト

ポートフォリオサイトのHeroセクションに、3Dキューブ型ナビゲーションを配置する。

目的は、最初の1画面だけで

-   どんな開発者なのか
-   どんな作品があるのか
-   どこへ移動できるのか

を伝えること。

------------------------------------------------------------------------

## デザインコンセプト

### テーマ

-   ダークテーマ
-   ブラックをベースにした近未来デザイン
-   青・紫・シアンをアクセントカラーに使用
-   ガラスUI（Glassmorphism）を部分的に採用
-   余白を活かしたシンプルなレイアウト
-   「Webアプリのようなポートフォリオ」を目指す

参考イメージ

-   Vercel
-   Linear
-   Apple
-   Raycast

------------------------------------------------------------------------

## Hero構成

### 左側

    Web Developer

    ユーザーの課題を
    Webアプリで解決する

    WordPress制作から
    Next.js・TypeScriptを使った
    Webアプリ開発まで対応。

    使いやすいUIと
    実用的なシステム開発を得意とする。

ボタン

-   View Projects
-   Contact

------------------------------------------------------------------------

### 中央

3Dキューブを配置する。

各面をクリックすると回転し、対応するセクションへ切り替える。

-   PROJECTS
-   SKILLS
-   ABOUT
-   CONTACT

初期表示は PROJECTS。

ドラッグ操作は後から追加し、まずはクリック操作のみ実装する。

------------------------------------------------------------------------

### 右側

操作説明

    Cube Navigation

    クリックで
    ページを切り替え

    現在選択中の
    セクションを表示

------------------------------------------------------------------------

## セクション定義

-   Projects（制作実績）
-   Skills（スキル・技術）
-   About（プロフィール）
-   Contact（お問い合わせ）

------------------------------------------------------------------------

## アニメーション

-   キューブはゆっくり浮遊する
-   面の切り替えは滑らかに回転
-   カードはフェードイン
-   ホバー時に軽く浮き上がる

派手すぎる演出は避ける。

------------------------------------------------------------------------

## 実装方針

1.  Heroを完成させる
2.  キューブを表示する
3.  ボタンクリックで回転
4.  セクションを切り替える
5.  レスポンシブ対応
6.  ドラッグ操作は後から追加
